# deep_retina_pytorch
Reimplementation of deep_retina in Pytorch

[Paper](https://arxiv.org/abs/1702.01825) and original [code](https://github.com/baccuslab/deep-retina) in Keras

## Requirements
* Pytorch
* tensorboardx
* numpy
* matplotlib and seaborn
